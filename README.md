# Assignment_11


Basic Operations and Data Structures in Python

This project demonstrates simple Python programs that perform basic arithmetic calculations and implement fundamental data structures.

Arithmetic Operations

The following operations are included:

Addition (Sum) Subtraction (Difference) Multiplication Division Data Structures

The project also covers basic data structures:

Stack – follows Last In, First Out (LIFO) principle Queue – follows First In, First Out (FIFO) principle Tuple – an immutable collection of elements Dictionary – stores data in key-value pairs All programs are implemented using Python, making it easy to understand core programming concepts and data handling techniques.
